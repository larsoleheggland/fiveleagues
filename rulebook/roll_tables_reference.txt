# Roll Tables Reference

This file catalogs all random tables in Five Leagues from the Borderlands for quick reference. Detailed table contents are in the relevant split files.

## Character Creation Tables
- Character Origin Table (D100 or choose) - determines race/origin
- Frontier Background Tables (D20 x4) - Capabilities, Mentality, Possessions, Training
- Mystic Background Tables (D20 x4)
- Noble Background Tables (D20 x4)
- Outsider Background Tables (D20 x4)
- Townsfolk Background Tables (D20 x4)
- Zealot Background Tables (D20 x4)
- Skills Table (D100) - random skill selection
- Follower Table (D100) - follower type
- Avatar History Table (D100) - starting bonus

## Campaign Setup Tables
- Settlement Population Table (D100) - number of hamlets/villages/towns
- Settlement Type Table (D100, modified by size) - settlement characteristics

## Preparation Stage Tables
- Town Events Table (D100) - local events in town
- Camp Events Table (D100) - events while camping
- Upkeep Costs Table (warband size) - Gold Marks owed
- Blacksmith Table (item type vs settlement size) - repair costs
- Foraging for Herbs Table (D6) - herb type found
- Gambling Table (D6) - gambling outcomes
- Hunting Expedition Table (D6) - hunting results
- Meet the Locals Table (D100, by settlement size) - social outcomes
- Connections Table (D100) - research connections
